# Michael Portfolio (React + Vite)

## Run locally
1) Install dependencies
```bash
npm install
```

2) Start dev server
```bash
npm run dev
```

## Customize
- Profile data: `src/data/profile.js`
- Projects data: `src/data/projects.js`
- Global styles: `src/styles/globals.css`

## Add your assets
- Put your photo in `public/avatar.jpg`
- Put your resume in `public/Michael_Massey_Resume.pdf`
